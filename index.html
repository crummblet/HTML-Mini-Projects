Welcome to my HTML Mini Projects

This repository contains a collection of HTML mini projects that I've made throughout my coding journey.

Some projects were created outside of the freeCodeCamp curriculum. These projects are my own work and are **not** labelled.

Some projects were created as part of the freeCodeCamp curriculum, where the goal was to build a project with similar functionality to an example project while adding my own design and content. These modules also allowed additions and improvements beyond the example. These projects are my own work and are **not** labelled.

Projects with the "_fCC" suffix are different. They were completed by following freeCodeCamp's step-by-step workshop lessons, where the instructions and almost all of the visible project content (not the code itself) were provided as part of the course. These are included to document my learning journey but are distinguished from my other projects.

The "_fCC" suffix is used to distinguish guided workshop exercises from projects where I created my own design and content. Enjoy!
