Welcome to my HTML Mini Projects

This repository contains a collection of HTML mini projects that I've created throughout my coding journey.

Some projects were created independently outside of the freeCodeCamp curriculum. Others were completed as part of the freeCodeCamp curriculum, where the objective was to build a project with functionality similar to an example while creating my own design and content. These projects often included additional features and improvements beyond the example. All of these projects are my own work and are not labelled.

Projects with the "_fCC" suffix are different. They were completed by following freeCodeCamp's step-by-step workshop lessons, where the instructions and most of the visible project content (not the code itself) were provided as part of the course. In some cases I wrote the project content myself, but because the project was still built by following the guided workshop, it includes the "_fCC" suffix.

The "_fCC" suffix is used to distinguish guided workshop exercises from projects where I independently created the design and content.

Enjoy!
